# Fluid Dynamics Solver

### Introduction
This research study deals with a square cavity, whose upper lid is moving with velocity “v”. This is a very popular single-phase problem in CFD which is generally used to test customized solvers. It involves the flow of an incompressible fluid inside the cavity driven by the tangen*al movement of the top wall, while the remaining walls are sta*onary (having no slip boundary condi*on).
