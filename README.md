# Incompressible Flow Solver

### Introduction
This research study deals with a square cavity, whose upper lid is moving with velocity “v”. This is a very popular single-phase problem in CFD which is generally used to test customized solvers. It involves the flow of an incompressible fluid inside the cavity driven by the tangen*al movement of the top wall, while the remaining walls are stationary (having no slip boundary condition).

### Code Implementation
To run the flow solver one needs to run the script run.sh. A detailed explanation of each python file is mentioned in the report along with the results.
